# 🛍 Retail Sales Data Analysis

This repository contains a retail sales analysis project performed on a dataset of ~50,000 transactions (2023–2024).  
The project uses **Python, Pandas, Seaborn, and Statistical Analysis** to generate insights.

---

## 📂 Files
- `retail_sales.ipynb` — Jupyter Notebook (full analysis)  
- `retail_sales.pdf` — Final Report (PDF)  
- `retail_sales_50k.csv` — Dataset (50k rows)  

---

## 📊 Key Insights
- **Top Categories**: Electronics, Sports, and Fashion (~60% revenue)  
- **Underperforming**: Home & Groceries  
- **Customer Behavior**: Female customers show slightly higher revenue  
- **Seasonality**: Sales peak during Nov–Dec (festive season)  

---

## ▶️ How to Run
```bash
pip install pandas numpy matplotlib seaborn scipy
```
